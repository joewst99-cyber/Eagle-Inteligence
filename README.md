# Eagle Intelligence

Live AI market intelligence terminal for Gold, Indices, and Forex.

## Deploy in 5 minutes

### Step 1 — Upload to GitHub
1. Go to github.com → New repository
2. Name it `eagle-intelligence`
3. Upload all these files (drag and drop the folder)

### Step 2 — Deploy on Vercel
1. Go to vercel.com → Sign up with your GitHub account
2. Click "Add New Project"
3. Select your `eagle-intelligence` repository
4. Click Deploy — done

Your live URL will be: `https://eagle-intelligence.vercel.app`

## What it does
- Live AI market intelligence powered by Claude + web search
- 9 assets: XAU/USD, S&P 500, Nasdaq, US30, DXY, EUR/USD, GBP/USD, GBP/JPY, USD/JPY
- Auto-refresh every 5, 10, or 15 minutes
- Deep dive per asset with key levels and correlation context
- Bias table — all assets at a glance
- Economic calendar with AI context
- Sentiment gauge (Risk-On / Risk-Off)
